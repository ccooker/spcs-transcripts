</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser TipTap editor → API | HTML content saved via authenticated PUT; rendered only in editor and exported server-side to PDF |
| Browser → POST /export | JWT Bearer via apiFetch; blob download same-origin |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-04-01 | Tampering | TipTap HTML stored client-side | accept | HTML rendered in PDF server-side only; no dangerouslySetInnerHTML of transcript HTML in list views |
| T-05-04-02 | Spoofing | Export PDF fetch | mitigate | apiFetch attaches Bearer token; no unauthenticated download URL |
| T-05-04-SC | Tampering | TipTap packages pre-installed | accept | Installed in Plan 01 with legitimacy audit |

</threat_model>
