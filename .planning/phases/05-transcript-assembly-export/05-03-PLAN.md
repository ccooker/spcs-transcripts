**As a** careers staff member, **I want to** open any student's transcript, edit the narrative sections, set the status to Draft or Finalised, and export a branded PDF, **so that** I can produce a complete, professional transcript in a single session without hunting through spreadsheets or paper records.

<objective>
Mount the transcript and settings API routes so the 12 RED integration tests from Plan 01 turn GREEN. Purpose: The service layer exists but routes return 404 — this plan completes the server vertical slice so the client (Plans 04–05) and integration tests have real HTTP endpoints.

Output: Two new routers, two Zod schema files, mounts in students.ts and app.ts, branding directory created on startup.
</objective>
