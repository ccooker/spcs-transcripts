## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → POST /api/students/:id/documents | Multipart upload with file + typeTag crosses here; file content and field values are untrusted |
| POST handler → disk | Buffer written to bound filesystem path; must be under UPLOAD_ROOT/students/{uuid} only |
| GET /:docId/download → browser | File streamed from disk; must be the authenticated student's file only |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | POST /documents fileFilter + route handler | mitigate | multer fileFilter rejects non-application/pdf MIME before buffering; magic bytes check (0x25 0x50 0x44 0x46) in route handler after buffering — dual-layer per D-06 |
| T-04-02 | Denial of Service | multer limits.fileSize | mitigate | limits.fileSize: 25*1024*1024 terminates upload at 25 MB per D-07 |
| T-04-03 | Elevation of Privilege | getDocumentForDownload / softDeleteDocument IDOR guard | mitigate | service checks doc.studentId === studentId before returning/deleting; rejects if deletedAt != null (consistent with Phase 3 IDOR pattern) |
| T-04-04 | Tampering | documentTypeTagSchema | mitigate | z.nativeEnum(DocumentType) rejects any string not in {REPORT_CARD, CERTIFICATE, AWARD_LETTER, WORK_EXPERIENCE_LETTER, REFERENCE_LETTER, OTHER}; validated in POST handler before any write |
